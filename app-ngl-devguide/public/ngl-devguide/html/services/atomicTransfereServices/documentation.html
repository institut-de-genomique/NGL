Module: atomicTransfereServices

Description:

  • Ce module contient plusieur service permettant la gestion des entrers/sortie d'experience.
    • oneToOne($scope, interface entrée, interface sortie)
    • manyToOne($scope, interface entrée, interface sortie)
    • oneTMany($scope, interface entrée, interface sortie)

Exemple d’utilisation:

  • $scope.atomicTransfere = oneToOne($scope,"datatable", "none");

Création de service:

  • Chaque service doit implementer ces fonctions en s'appuyant sur les services oneToX, XToOne, ManyToX, XToMany:

    • experimentToInput: Cette fonction doit passer les données de la vue des entrées au model $scope.experiment.value
    • inputToExperiment: Cette fonction doit passer les données du model ($scope.experiment.value) à la vue des entrées
    • outputToExperiment: Cette fonction doit passer les données de la vue des sorties au model $scope.experiment.value
    • experimentToOutput: Cette fonction doit passer les données du model ($scope.experiment.value) à la vue des sorties
    • loadExperiment: Cette fonction doit charger les données d'une experience existante
    • newExperiment: Cette fonction doit crée une nouvelle experience
