Package: views.helper

Description:

  • Ce module fourni des helpers play pour Twitter Bootstrap, il permet aussi d'utiliser les derniers standards HTML5

Exemple d'utilisation:

  • @helper.tbInputText(monFormulaire("field"),'btstyle->"small",'addonb->"Nom")

Liste des helpers:

  • tbCheckBox(Field,args) (génère une checkbox)

    • Options possibles dans args:

      • 'states = enabled ou disabled (default enabled) état de la checkbox active ou pas
      • 'inline = enabled ou disabled (default disabled) affiche les checkBox en ligne ou pas
  • tbInput(Field,type :String,args) (génère un input de type="type") :

    • Options possibles dans args:

      • 'states = enabled ,disabled (default enabled)
      • 'addonb = String affiche un addon avant l'input
      • 'addona = String affiche un addon après l'input
      • 'btstyle = String (defini la taille de l'input): mini,small,medium,large,xlarge,xxlarge (default medium)
      • 'bticonaddonb = String (ajoute l'icône dans l'addon avant l'input) nécessite 'addonb
      • 'bticonaddona = String (ajoute l'icône dans l'addon après l'input) nécessite 'addona

    • Input qui utilise cette methode:

      • tbInputText
      • tbInputDate
      • tbInputFile
      • tbInputNumber
      • tbInputMail
      • tbInputColor
      • tbInputPassword
      • tbInputUrl
      • tbInputTel
      • tbInputRange
  • tbInputRadio(Field, options: Seq[(String, String)],args)

    • Options possibles dans args:

      • 'states = enabled ou disabled (default enabled) état de la checkbox active ou pas

    • Exemple d'utilisation:

      • @helper.tbRadio(monFormulaire ("field"), Seq("M"->"Male","F"->"Female"))
        Ce code génère deux radiobox du même groupe (ici field), une avec la valeur M et le label Male et l'autre avec la valeur F et le label Female
  • tbTextArea(Field,args)

    • Options possibles dans args:

      • 'states = enabled ,disabled (default enabled)
      • 'addonb = String affiche un addon avant l'input
      • 'addona = String affiche un addon après l'input
      • 'bticonaddonb = String (ajoute l'icône dans l'addon avant l'input) nécessite 'addonb
      • 'bticonaddona = String (ajoute l'icône dans l'addon après l'input)
  • tbSelect(Field, selectedValue: List[String], options:Map[String,String],args)

    • Options possibles dans args:

      • 'states = enabled ,disabled (default enabled)
      • 'multi = enabled or disabled (default disabled) enabled = le select est multi-choix
  • tbInputUneditable(Field,args)

    • Options possibles dans args:

      • 'states = enabled ,disabled (default enabled)
      • 'addonb = String affiche un addon avant l'input
      • 'addona = String affiche un addon après l'input
      • 'btstyle = String (defini la taille de l'input): mini,small,medium,large,xlarge,xxlarge (default medium)
      • 'bticonaddonb = String (ajoute l'icône dans l'addon avant l'input) nécessite 'addonb
      • 'bticonaddona = String (ajoute l'icône dans l'addon après l'input) nécessite 'addona
      • 'hidden = yes ou no
  • tbformhorizontal et tbformvertical(elements: FieldElements)
    • Balises qui remplacent la balise form de play et qui permet de préciser l'alignement des champs
