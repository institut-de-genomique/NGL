Package: play.modules.cas

Description:

  • Ce plugin permet de gérer l’authentification avec CAS

Exemple d’utilisation:

  • En annotation devant une méthode: @With(CasAuthentication.class)

Configuration:

  • Dans le fichier Application.conf il faut définir les options :

    • casUrlValidator = "https://cas.genoscope.cns.fr:8443/cas/serviceValidate" url de validation
    • casUrlLogin = "https://cas.genoscope.cns.fr:8443/cas/login" url de login
    • casRenew = false ou true si renew est true a lors l’utilisateur devras se connecter à chaque entrée dans l’application
    • casMode = “debug” ou “prod” si mode = debug le plugin ne verifie pas d'information et crée un cookie avec un utilisateur fictif
    • casTimeOut = durée de vie du cookie d'authentification en milliseconde (3600000 par defaut)
