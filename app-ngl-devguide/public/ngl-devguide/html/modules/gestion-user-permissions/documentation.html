Package: controllers.authorisation

Description:

  • Cette annotation/helper permet de gérer les authorisations et l'accès aux différentes ressources.

Exemple d’utilisation:

  • En annotation devant une méthode: @Permission(value={"54ki2","test"},teams={"THETEAM","team2"})
  • Dans un template:
    • @if(PermissionHelper.checkPermission(session,"code permission")) { @* instructions *@} (session est un mot-clé scala)
    • Avec une liste de permission à verifier, allPermission permet de demander à ce que l'utilisateur ai toute les permissions(true) ou juste une des permissions(false)
      @if(PermissionHelper.checkPermission(session,Array(String)permissions,String allPermission)) { @* instructions *@} (session est un mot-clé scala)
    • @PermissionHelper.checkTeam(Session ses, String varteam){ @* instructions *@} (ses est un mot-clé scala)
    • @PermissionHelper.checkTeam(Session ses, Array(string) varteam){ @* instructions *@} (ses est un mot-clé scala)
