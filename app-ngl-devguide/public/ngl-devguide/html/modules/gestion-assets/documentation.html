Package: controllers.resources

Description:

  • Cette annotation/helper permet de gérer les authorisations et l'accès aux différentes ressources.

Exemple d’utilisation:

  • Ceci générera l'url http://serveur/stylesheets/bootstrap/2.1.1/bootstrap.css : href="@helper.getAsset("css","bootstrap","2.1.1","bootstrap_2.1.1.css")"

Configuration:

  • Dans le fichier Application.conf il faut définir les options :

    • asset.url = url du serveur de ressource
