Information importante

Il existe 2 modes d'utilisation pour importer les profils bioanalyzer :

  • le mode "semi-automatique"
    1. Renseigner la position de chaque container sur la puce via la colonne position (positions possibles : 1 à 12). Ne pas renseigner 2x la même position. L'expérience doit contenir au maximum 12 containers. SAUVEGARDER
    2. Puis, via l'onglet Profils, importer les (1 à) 12 images (.JPEG ou .PNG, pas d'autres formats). Les noms des images doivent respecter cette nomenclature : xxxxxxx_SampleN (N = 1 à 12). SAUVEGARDER. Vérifier que les images se sont bien insérées dans le tableau des containers.
  • le mode "manuel"
    • Dans ce cas il n'est PAS nécessaire de renseigner la colonne position.
    • Choisir l'image à importer de chaque container, ligne par ligne, via la colonne Profil de migration
    • S'assurer que les images sont au format .JPEG ou .PNG (pas d'autres formats)
